Microsoft Visual Studio

Required: Windows XP Service Pack 2

Visual Studio requires that Windows XP Service Pack 2 be installed on your computer before you install Visual Studio. You can download and install this Service Pack from http://windowsupdate.microsoft.com/. The Internet Explorer 6.0 Service Pack 1 is included in the Windows XP Service Pack 2 as well. After you install this Service Pack, you are required to restart your computer. Visual Studio setup will not automatically restart. To restart Visual Studio setup, double-click setup.exe on the CD, DVD, or network share you installed from or click the appropriate link on the download Web site.

Note: It is highly recommended that you regularly check for and install updates for your operating system. To search for operating system updates, see http://windowsupdate.microsoft.com/.

Why install Service Packs?

Service Packs are the means by which product updates are distributed. Service Packs can contain updates to improve system reliability, program compatibility, security, and more. For more information on Microsoft Service Packs, see http://support.microsoft.com/default.aspx?scid=fh;EN-US;sp.
